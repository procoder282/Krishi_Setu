# 🌾 Krishi Setu

Krishi Setu is an AI-powered smart agriculture platform designed to support farmers with intelligent crop disease detection and early diagnosis. By leveraging **Deep Learning** and **Computer Vision**, the system analyzes plant leaf images and provides accurate, real-time disease predictions, helping farmers take timely action to improve crop health and reduce losses.

## 🚀 Features

- ✅ AI-powered crop disease detection
- ✅ Simple and mobile-friendly user interface 📱
- ✅ Real-time disease diagnosis using Deep Learning
- ✅ Supports **38+ crop disease categories**
- ✅ Instant prediction with high accuracy
- ✅ Easy-to-use web application built with **Streamlit**

## 📷 How It Works?

1. 📸 **Capture** – Take a clear image of the crop leaf.
2. ⬆️ **Upload** – Upload the image through the Krishi Setu application.
3. 🤖 **Analyze** – The AI model processes the image using Deep Learning techniques.
4. 📊 **Results** – Instantly receive the predicted disease along with useful recommendations.

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **Backend:** Python
- **Machine Learning:** TensorFlow, Keras, NumPy
- **Model:** Convolutional Neural Network (CNN)
- **Deployment:** Streamlit Cloud / Local Server

## 📂 Dataset

- **Source:** Plant Diseases Dataset (Kaggle)
- **Images:** 87,000+ labeled crop leaf images
- **Classes:** 38+ plant disease categories
- **Data Augmentation:** Rotation, flipping, zooming, and scaling for improved model performance.

## 🌱 Vision

Krishi Setu aims to bridge the gap between modern AI technology and agriculture by providing farmers with an accessible, reliable, and intelligent crop health monitoring solution that promotes sustainable farming and improves agricultural productivity.
